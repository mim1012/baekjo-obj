# docs/archive

**용도**: 더 이상 현재 작업의 기준이 아닌 과거 계획·조사·핸드오프 문서의 보관처
(2026-07-18 docs 폴더 재구성으로 신설).

## 수정 금지 원칙 (append-only)

- 이 폴더의 기존 파일은 **내용을 고치지 않는다.** 여기 있는 문서는 "작성 당시의 결정·조사
  기록"으로서 가치가 있고, 사후에 손대면 그 기록성이 사라진다.
- 새 문서를 추가하는 것(append)은 언제나 괜찮다 — 과거 계획이 새로 완결되거나 폐기되면
  그 경위를 새 파일로 남기거나, 살아있는 SSOT(AGENTS.md·SESSION.md·memory)에 한 줄
  포인터를 남기고 원문은 그대로 둔다.
- 오탈자·깨진 링크 수정처럼 사소한 교정이 필요하면, 고치기 전에 왜 필요한지와 무엇을
  바꾸는지 남기고 진행한다(이 폴더로 옮겨오는 이번 재구성 커밋의 경로 수정 정도가 예외다).
