# Baekjo Platform Completion Implementation Plan

## Phase 1: Data and Storage Foundation

- [ ] 핵심 타입을 하위 호환 방식으로 확장
- [ ] 상품 분류와 보험 mock data 보완
- [ ] 장바구니 계산, 보험 메모, 주문 조회, mock 인증 함수 추가

## Phase 2: Customer Journey

- [ ] 헤더와 홈의 누락 섹션/링크 추가
- [ ] 쇼핑 필터·정렬·상품 카드 보완
- [ ] 상품 상세 Q&A/추천/브랜드 흐름 보완
- [ ] 주문완료 실제 데이터 표시
- [ ] 보험 동의/관심 보장 폼과 소개 문구 보완
- [ ] 로그인 저장, 회원가입, 마이페이지 메뉴 보완
- [ ] 후기/공지 컬럼과 브랜드 상세 후기 보완

## Phase 3: Admin Console

- [ ] 관리자 전용 셸과 전체 사이드바/상단바 구성
- [ ] 대시보드 통계/목록 보완
- [ ] 회원·브랜드·고민·주문·후기·Q&A·공지 관리 라우트 생성
- [ ] 상품/보험 관리 화면 보완

## Phase 4: Validation

- [ ] 필수 라우트 존재 확인
- [ ] ESLint 통과

- [ ] 핵심 localStorage 흐름과 반응형 화면 확인
